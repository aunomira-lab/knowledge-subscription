### 4. MVP栏目 (来自 mvp_spec.md)

**MVP范围**:
- 每日AI商机简报生成系统
- 用户订阅管理 (收集邮箱/WeChat ID)
- 静态销售页展示价值和定价
- 支付接入 (小报童快速启动)

**技术架构**:
- 前端: 静态HTML + Tailwind CSS
- 托管: Vercel / Cloudflare Pages
- 后端脚本: Python 3.11
- 数据库: SQLite / Notion API
- 邮件发送: Resend API
- AI生成: OpenAI GPT-4o-mini

**核心功能**:
1. **内容生成系统** (GEN-001): 每日简报自动生成
2. **内容模板系统** (GEN-002): 统一Markdown格式模板
3. **订阅收集** (SUB-001): SQLite数据库存储用户信息
4. **邮件发送** (SND-001): Resend API批量发送
5. **静态销售页** (WEB-001): 单页HTML展示产品价值
