## 1. 收费产品和盈利假设

- 早鸟版：¥29/月。每周 3-5 份机会简报，目标冷启动转化。
- 专业版：¥99/月。每日机会、脚本模板、Prompt、复盘表，目标主收入。
- 定制版：¥499/次。按用户行业/技能/账号资源做 1 份专属机会雷达，目标现金流和高客单验证。

- 首周收入目标：5 个早鸟 + 1 个定制 = ¥644；保守只转化 5 个早鸟 = ¥145。

4. 跨境/电商从业者：需要工具套利、内容批量化、客服自动化案例。
