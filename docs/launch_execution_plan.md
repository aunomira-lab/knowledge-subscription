1. 知乎：问题型长回答，强调“如何判断AI副业是不是伪需求”。

## 广告投放前置条件
投放前必须全部满足：
- GitHub Pages 或 Cloudflare Pages URL HTTP 200。
- 真实收款和客服入口可用，不只依赖占位符。
- 自然流量7天达到：1000曝光、50线索、5笔付费；或咨询转付费≥10%。
- CAC 预估 < ¥35（早鸟）或 < ¥120（Pro）。
- 页面声明“不保证收益，不构成投资/经营建议”。

## 收款与交付闭环
- 授权前：邮件/微信人工收线索，发送收款方式，人工确认付款，24h内发送首周内容包。
- 授权后：小报童/知识星球/Stripe/PayPal/微信收款链接替换 mailto CTA；购买后自动发欢迎邮件和首周包。
- 客服SLA：24h内响应；未交付可退，已交付数字内容按收款平台规则处理。

## 失败/Pivot阈值
- 7天曝光 < 500：标题/平台选择失败，换更垂直赛道，如“AI获客脚本订阅”。
- 线索率 < 2%：免费样例价值表达不足，重写首屏和样例。
- 5个高意向咨询无付款：价格或信任不足，改成¥9试读包或1对1定制诊断。
