## 5. 获客平台计划

1. 知乎：搜索“AI副业、AI自动化、独立开发、知识付费、Dify/Coze赚钱”，每天10条高相关回答，CTA 免费样例。
2. 小红书：发布卡片笔记，标题围绕“普通人能卖的AI服务”“AI客服报价单”“不是提示词，是交付SOP”。
3. 即刻/V2EX：以独立开发复盘和工具链透明化方式发布，避免硬广。
4. 微信群/朋友圈：先做人工预售，不等自动支付；用早鸟20席制造紧迫感。
5. Product Hunt/Indie Hackers（英文后续）：若中文转化>3%，将样例翻译为英文并开 Substack。

## 6. 广告投放前置条件

付费广告必须等待以下条件全部满足：
- 公开URL HTTP 200 且页面含 CTA、价格、退款边界、免责声明。
- 真实收款链接已替换 PAYMENT_URL_TO_FILL_AFTER_USER_AUTH。
- 客服入口可用并能 12h 首响。
- 至少3篇免费样例内容可访问。
- CSV或表格能记录 utm_source、访问、留资、订单、收入。
- 第一轮自然流量访问->留资 >= 5%，否则先不投广告。

## 7. 首周盈利测算

保守目标：
- 访问 600，留资 48（8%），付款 10（20%留资转化）。
- 8个早鸟 ¥29 + 2个专业版 ¥99 + 1个定制 ¥499 = ¥929。
- 主要成本为内容生成和人工客服，毛利率预计 >85%。

失败阈值：
- 访问->留资 <3%：销售页主张或免费样例不够强。
- 留资->付费 <10%：价格/信任/交付承诺需要调整。
- 首周收入 <¥300：优先推 ¥499 定制诊断，而不是只卖低价订阅。

## 8. 人工预售SOP

1. 用户邮件/私信：“我要样例/订阅”。
2. 回复免费样例+三档价格+付款方式。
3. 用户付款后记录到 metrics/customers.csv 或 payment_delivery_tracker.csv。
4. 24小时内发送首周内容包和下期交付时间。
5. D+3 询问执行反馈，D+7 推续费或定制诊断。
