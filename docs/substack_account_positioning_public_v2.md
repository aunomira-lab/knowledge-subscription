# 账号定位修正版：大众人工智能知识型博主

更新时间：2026-05-10T02:52:31Z
项目：knowledge-subscription
版本：v2，大众向修正

## 1. 用户反馈后的关键修正

上一版把账号定位为偏开发者、独立开发者、产品和运营人群的“工作流实战周报”。这个方向过窄，也默认观众已经理解人工智能的价值。

用户明确纠正：首批观众不应该默认懂人工智能，也不应该默认知道人工智能能挣钱、能降本增效、能改变普通人的工作方式。很多普通观众对人工智能的认知仍停留在：它像一个会聊天的问答工具，可以回答很多问题，但有时候会胡说。

因此，账号定位必须从“专业从业者的工作流账号”改为：

面向普通大众的人工智能知识型博主，帮助普通人理解人工智能、使用人工智能、适应人工智能社会，并定期讲中美人工智能新闻、新工具和普通人的实际用法。

## 2. 新账号定位

中文定位：
人人都能听懂的人工智能知识博主。

一句话：
用普通人听得懂的话，讲清楚人工智能正在怎样改变生活、工作和赚钱方式，并教大家一步一步把人工智能用起来。

账号使命：
让大众全面拥抱人工智能，而不是被人工智能变化吓到。

## 3. 目标观众

第一层：普通上班族
- 不懂技术，但担心未来工作会被影响。
- 想知道人工智能到底能帮自己做什么。
- 关心提效、简历、办公、写作、学习、沟通、找副业。

第二层：个体经营者和小老板
- 想降低客服、文案、运营、销售、设计成本。
- 不懂工具连接，只想知道怎么马上用。
- 需要案例：一个小店、一个工作室、一个自由职业者怎么用。

第三层：学生和家长
- 关心学习、作业、考试、专业选择和未来职业。
- 需要理解人工智能时代要培养什么能力。

第四层：内容创作者和自由职业者
- 想用人工智能做选题、脚本、剪辑、资料整理、客户沟通。
- 需要低门槛、可模仿的流程。

## 4. 内容支柱

### 4.1 人工智能基础认知
目的：先消除陌生感和误解。

选题示例：
- 人工智能到底是什么？为什么它不只是聊天？
- 为什么人工智能有时会胡说？普通人怎么判断？
- 你不需要懂代码，也能用人工智能做什么？
- 人工智能真正改变的不是答案，而是做事流程。

### 4.2 中美人工智能新闻解读
目的：建立“跟着我能看懂趋势”的心智。

内容要求：
- 不堆新闻。
- 每条新闻必须解释：这和普通人有什么关系。
- 少讲机构名和技术名，多讲影响。

固定结构：
发生了什么 → 为什么重要 → 普通人该注意什么 → 有没有现在就能试的用法。

### 4.3 新工具介绍
目的：帮助普通人实际用起来。

要求：
- 中文版本尽量不出现英文名或国外工具名；如必须出现，只在字幕/备注里出现。
- 口播里用“写作工具、图片工具、办公工具、翻译工具、视频工具、资料整理工具”等普通人能懂的分类。
- 每次介绍只解决一个具体问题。

### 4.4 普通人工作流
目的：把“会问问题”升级成“会办事”。

示例：
- 用人工智能做一份每日信息简报。
- 用人工智能整理会议纪要。
- 用人工智能给小店写宣传文案。
- 用人工智能把客户问题整理成常见问答。
- 用人工智能辅助孩子做学习计划。

### 4.5 赚钱和降本增效案例
目的：建立商业价值，但不能上来就讲暴富。

表达方式：
- 不说“人工智能让你马上赚钱”。
- 说“它先帮你省时间、省人工、省试错”。
- 再解释：省下来的时间和成本，如何慢慢变成更多收入机会。

## 5. 视频口吻规则

必须做到：
- 可以使用“AI”这个大众已熟悉的词，不必每次都说“人工智能”。
- 开头先解释背景，不默认观众懂 AI 的真实价值。
- 少用英文，中文口播不要堆英文名；如必须提工具，优先提中国内地用户可直接使用的工具。
- 具体例子必须能落地：说明用什么国内工具、复制什么话术、粘贴到哪里、得到什么结果。
- 不停留在提示词技巧：要说明这是入门层，后续会讲工具使用、多工具串联、半自动流程等更高阶玩法。
- 不提复杂技术名，不堆国外工具。
- 例子必须贴近普通人：上班、开店、写文案、做表格、回客户、学东西。
- 制造焦虑要克制：讲变化正在发生，不恐吓。
- 像专业大佬聊天，不像课程销售页。

避免：
- “不会用就被淘汰”。
- “一键赚钱”。
- “人人都能副业起飞”。
- “这个工具太强了”。
- “颠覆所有行业”。
- 一上来讲开发者、接口、自动化脚本、国外平台。

## 6. 首支视频新方向

主题：
很多人以为人工智能只是回答问题，其实它真正的价值是帮普通人少走弯路、少做重复劳动。

核心工作流：
用人工智能做一个“每日三件事助手”。

受众最广，因为每个人每天都要面对信息、任务和选择。

流程：
1. 把今天看到的信息和要做的事告诉人工智能。
2. 让它帮你分成：必须做、可以等等、可以不管。
3. 让它把必须做的事拆成第一步。
4. 晚上让它帮你复盘：今天什么事真正有价值。

这个工作流比“信息过滤器”更大众，适合第一期。

## 7. 账号名称建议

中文主名优先：
- 人人都能用人工智能
- 普通人的人工智能课
- 人工智能生活指南
- 跟我用人工智能
- 人工智能时代生存指南

推荐：普通人的人工智能课

原因：
- 不高冷。
- 目标用户明确。
- 适合新闻、工具、案例、教程长期展开。
- 不像单纯卖课，也不像技术号。

副标题：
讲中美人工智能新闻，拆实用工具，教普通人适应人工智能社会。

## 8. 对旧文档的处理

旧的“面向开发者、产品、运营的工作流周报”可以作为后续高阶付费栏目，不再作为首期公开视频定位。

当前公开视频和涨粉内容，必须先做大众教育：
先讲懂，再讲会用；先讲省时间和少踩坑，再讲收入机会。
