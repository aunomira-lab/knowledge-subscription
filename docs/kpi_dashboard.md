## 二、转化漏斗实时看板

```
┌───────────────────────────────────────────────────────────────────────────────────────────────┐
│  曝光层    │  访问层    │  试看层    │  意向层    │  付费层    │
│  500/日    │  50/日     │  10/日     │  3/日      │  1/日      │
│  基准:500  │  基准:50   │  基准:10   │  基准:3    │  基准:1    │
│  ───────────│  ───────────│  ───────────│  ───────────│  ───────────│
│  转化率    │  10%      │  20%      │  30%      │  33%      │
│  健康基准  │  ≥10%     │  ≥20%     │  ≥8%      │  ≥3%      │
└───────────────────────────────────────────────────────────────────────────────────────────────┘
```

### 漏斗健康诊断

| 阶段 | 健康 | 预警信号 | 危险信号 |
|------|------|----------|----------|
| 曝光→访问 | ≥10% | 5-10% | <5% |
| 访问→试看 | ≥20% | 10-20% | <10% |
| 试看→意向 | ≥8% | 3-8% | <3% |
| 意向→付费 | ≥3% | 1-3% | <1% |
| 付费→激活 | ≥70% | 50-70% | <50% |
| 激活→续费 | ≥50% | 30-50% | <30% |

**诊断逻辑**: 当任一阶段低于危险信号时，立即锁定该阶段进行优化，其他阶段暂停调整。

## 四、周度复盘模板

### 每周日22:00执行

| 复盘项 | 内容 |
|--------|------|
| 本周收入 | 累计收入、日均收入、环比增长 |
| 漏斗表现 | 各阶段转化率对比上周 |
| 渠道表现 | 各渠道获客成本、转化率、ROI |
| 内容表现 | 阅读量最高/最低内容、用户反馈关键词 |
| 用户反馈 | 正面评价、负面评价、功能需求 |
| 绕过收款表现 | 各绕过方案收入、次数、每笔均价 |
| 下周计划 | 调整内容主题、渠道、定价策略 |

### 周度关键问题

1. 本周是否达成收入目标？若未达成，瓶颈在哪个漏斗阶段？
2. 哪个渠道ROI最高？是否应加码投入？
3. 哪篇内容反响最好？能否复制其结构/主题？
4. 用户最常问的问题是什么？是否需要更新FAQ？
5. 是否有投诉或退款？根因是什么？
6. 绕过收款方案哪个效率最高？是否需要调整比例？

## 五、预警指标

### 预警分类

| 级别 | 级别代码 | 定义 | 响应时限 |
|------|------|------|------|
| 红色 | RED | 收入渠溏为0、支付中断、投诉漫增 | ≤1h |
| 橙色 | ORANGE | 转化下降、打开率下降、获客成本超标 | ≤24h |
| 黑色 | BLACK | 收入停滞超过3天 | 直即停止新付费推广，全面复盘 |

### 红色预警（立即处理）

| 预警 | 触发条件 | 响应动作 | 升级路径 |
|------|----------|----------|----------|
| 收入停滞 | 连续3天收入=0 | 检查支付通道+销售页可用性 | 2h内未解决→通知小电 |
| 支付故障 | 用户报告无法支付 | 测试支付流程+切换备用通道 | 1h内未解决→通知小电 |
| 内容中断 | 连续2天未发布内容 | 启动紧急内容生产 | 当日必须补上 |
| 投诉激增 | 单日投诉≥3 | 暂停新付费入口，排查问题 | 立即通知小电 |
| 退款潮 | 单日退款≥2 | 暂停销售，启动退款根因分析 | 立即通知小电 |
| 收款码失效 | 微信/支付宝收款码被封或不可见 | 立即更换二维码图片并检查链接 | 立即通知小电 |

### 橙色预警（24h内处理）

| 预警 | 触发条件 | 响应动作 |
|------|----------|----------|
| 转化率下降 | 周转化率下降>30% | 检查销售页+定价策略 |
| 打开率下降 | 邮件打开率<30%连续3天 | 优化标题+发送时间 |
| 获客成本上升 | CAC>¥50 | 优化渠道或暂停付费投放 |
| 退订率上升 | 周退订率>10% | 调查退订原因+优化内容 |
| 绕过收款下降 | 连续3天绕过收款=0但流量正常 | 检查二维码可见性+收款码贴图位置 |

## 七、健康度评分卡

### 每日评分（满分100）

| 维度 | 权重 | 得分 | 计算方式 |
|------|------|------|----------|
| 收入健康 | 30 | 0 | 达成日目标=30，未达成按比例 |
| 漏斗健康 | 25 | 0 | 各阶段转化率加权平均 |
| 留存健康 | 20 | 0 | 留存率 vs 目标 |
| 运营健康 | 15 | 0 | 内容按时发布+响应及时 |
| 风险健康 | 10 | 10 | 无投诉退款=10 |
| **总分** | **100** | **10** | 冷启动期基准分 |

**评分解读**:
- 90-100: 优秀，可加码
- 70-89: 健康，保持观察
- 50-69: 预警，需优化
- <50: 危险，考虑可止
