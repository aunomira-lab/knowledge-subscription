## 一、商业定位

### 1.1 一句话定位

> **AI 开发者赚钱情报：每天 5 分钟，掌握可落地的 AI 变现线索 + 可执行代码/SOP。**

### 1.2 核心价值主张

| 维度 | 竞品痛点 | 我们的解法 |
|------|---------|-----------|
| 信息过载 | 用户每天被 100+ AI 资讯轰炸 | 只筛选「能赚钱」的 3-5 条，附执行步骤 |
| 落地困难 | 看到机会但不知道怎么做 | 每条附带「可执行 SOP + 代码/提示词模板 + 收益测算」 |
| 时效性差 | 周报/月报错过窗口期 | 每日更新，窗口期优先推送 |
| 语言壁垒 | 英文内容多，中文开发者看不懂 | 中文解读 + 可运行代码/JSON 模板 |
| 内容生产 | 个人创作者产能有限 | AI 自动化收集 + 人工精选审核，质量门禁 88.2% |

### 1.3 差异化定位地图

```
高定价
    |
¥500+/月 │   知识星球 (高端社群) / 头号王牌猎人 ¥299-999/年
    │   SemiAnalysis $200-500/月
    |
¥100-500/月│   专业版 (¥99/月) ★ 我们在这里
    │   高级版 (¥299/月) ★ 我们在这里
    │   The Pragmatic Engineer $15/月
    |
¥30-100/月 │   早鸟版 (¥29/月) ★ 我们在这里
    │   小报童 / Substack / 魔乐谷 $10/月
    |
低定价    │   公众号付费阅读 / 知乎盐选
    │
    └──────────────────────────────────────
         低信息密度    高信息密度
              低可执行性    高可执行性
```

**关键差异**: 中文市场唯一「数据验证 + SOP + 可运行代码 + 收益测算」组合的付费简报。

## 三、内容定位

### 3.1 内容栏目 (MVP 版)

| 栏目 | 说明 | 字数 | 付费门槛 |
|------|------|------|---------|
| **头条机会** | 当天最值得关注的 1 个 AI 变现机会 | 300-500字 | 免费试读 |
| **3个今日线索** | 附带执行步骤的 3 条可落地机会 | 各200字 | 付费 |
| **数据信号** | 投融资、产品发布、流量变化等量化信号 | 200字 | 付费 |
| **工具速递** | 新上线的可变现 AI 工具 + 试用链接 | 150字 | 付费 |
| **执行模板** | 可复制的工作流/脚本/提示词/JSON 模板 | 附文件 | 专业版/高级版 |
| **社区精选** | 订阅者分享的实战经验 | 200字 | 专业版/高级版 |

### 3.2 内容生产流程

```
每日 06:00 │ AI 自动抓取 20+ 数据源（RSS/Reddit/HN/Product Hunt）
每日 08:00 │ 初筛 50 条 → 精选 10 条
每日 09:00 │ 人工审核 + 补充执行步骤 + 代码/SOP 模板
每日 10:00 │ 生成 Markdown/HTML 简报
每日 11:00 │ 推送至邮件列表 + 小报童/知识星球
```

### 3.3 内容质量红线（四道门 + 禁用词）

| 检查项 | 标准 | 责任 |
|--------|------|------|
| 可执行性 | 必须附带至少 3 个可执行步骤 | 内容审核 |
| 时效性 | 信息不得超过 48 小时 | 数据采集 |
| 真实性 | 必须提供可验证的来源链接 | 数据审核 |
| 价值密度 | 单条信息 ROI 预估需明确 | 内容审核 |
| 版权合规 | 摘录保留来源，不搬运全文 | 法务检查 |
| 禁用词 | 禁止"稳赚""无脑""今晚就能试""躺赚"等低端词汇 | AI 初筛 + 人工复核 |
