## 1. 一句话定位

> **"AI赚钱机会雷达——每天发现一个可执行的AI变现机会，附带脚本、Prompt和收益测算"**

**为什么能收钱**：

- 痛点极痛：AI工具每天更新，但普通人不知道哪些机会是今天能上手干的

**海外升级路径（M3-M6验证后启动）**：AI Architecture Weekly（ADR格式，$15/mo，对标 Lenny's Newsletter）

## 2. 目标付费用户

### 主力 Persona：副业探索者 小李
- **画像**：25-35岁，互联网从业者/自由职业者，月收入1-3万
- **痛点**：想搞副业但不知道AI时代做什么、怎么做、能赚多少
- **付费预算**：¥29-99/月（决策门槛低，感知为"一杯咖啡"）
- **现有信息源**：微信公众号、小红书、知识星球（信息过载、缺乏执行细节）
- **转化钩子**："今天花5分钟看完，周末就能上手跑通第一个MVP"

### 次要 Persona：技术变现者 老张
- **画像**：独立开发者、全栈工程师，有技术能力但缺商业方向
- **痛点**：不想接外包，想用AI做产品但不知道什么方向有真实付费需求
- **付费预算**：¥99-299/月
- **转化钩子**：MCP Server、SaaS模板、自动化工作流——带代码和部署步骤

### 第三 Persona：小团队老板 王总
- **画像**：5-20人小团队负责人，想用AI降本增效
- **痛点**：没时间研究AI工具，需要经过验证的解决方案清单
- **付费预算**：¥499-999/年
- **转化钩子**：团队版授权 + 季度AI降本审计报告

## 3. 内容定位与栏目

### 核心付费产品：每日「AI赚钱机会简报」

**格式**：
1. 机会标题 — 一句话说明做什么、预期收益区间
2. 分类标签 — AI基础设施 / 自动化 / 社媒变现 / 跨境电商 / 内容创作
3. 难度星级 — 1-5星，匹配用户能力
4. 启动时间 — "3天" / "1周" / "2周"
5. 收益测算 — 具体到平台的保守计算公式
6. 执行SOP — 每一步的工具、命令、Prompt
7. 数据来源 — GitHub stars、Reddit讨论量、平台政策链接
8. 风险提醒 — 合规、封号、竞争饱和度

**长度**：3-5分钟阅读，800-1,500字/期

### 内容支柱（按周轮换）

| 星期 | 支柱 | 示例选题 |
|------|------|---------|
| 周一 | AI基础设施 | MCP Server电商数据服务 |
| 周二 | 自动化/效率 | n8n工作流模板商店 |
| 周三 | 社媒变现 | 小红书AI穿搭账号矩阵 |
| 周四 | 内容创作 | AI短剧批量生成+分发 |
| 周五 | 跨境/出海 | Temu半托管模式套利 |
| 周六 | 工具评测 | 本周新出AI工具测评+变现路径 |
| 周日 | 复盘/案例 | 一个真实跑通的案例拆解 |

| 竞品 | 他们的模式 | 我们填补的空隙 |

## 4. 定价与套餐

### 套餐设计

| 套餐 | 价格 | 内容 | 目标量(M6) | 收入影响 |
|------|------|------|-----------|----------|
| 免费试看 | ¥0 | 每周1期节选（3个机会摘要） | 2,000人 | 线索获取 |
| 早鸟版 | ¥29/月 | 每日1期完整简报（2-3个机会） | 200人 | ¥5,800/月 |
| 专业版 | ¥99/月 或 ¥799/年 | 每日2-3期 + 执行脚本 + Prompt模板 + 会员群 | 50人 | ¥4,950/月 |
| 创始会员 | ¥199/年（前100名） | 终身早鸟价锁定 + 创始人徽章 + 1v1咨询1次 | 100人 | ¥19,900一次性 |

### 定价依据
- 小报童头部作者已验证¥29-99/月价格带可行
- 中国市场60%用户愿为优质内容付费；LTV/CAC 22-84:1
- ¥29/月 = 一杯咖啡，感知门槛低，12个月LTV ¥348
- ¥199/年创始会员制造稀缺性，一次性回收内容生产成本

### 收入模型
- **主业**：订阅复购（毛利率 85%+，数字内容边际成本趋近于零）
- **副业**：工具模板 upsell（¥49-199一次性，如n8n工作流、MCP Server源码）
- **第三收入**：咨询线索（¥500-2000/次，帮用户评估AI落地可行性）

## 7. 风险登记与缓解

| 风险 | 可能性 | 影响 | 缓解措施 |
|------|--------|------|----------|
| 内容质量/真实性 | 中 | 高 | 每个机会必须有数据来源、收益区间而非画饼、标注难度 |
| 订阅增长慢于计划 | 中 | 高 | 加码小红书/知乎/即刻；销售页A/B测试；创始会员稀缺性 |
| 平台封号/限流 | 中 | 高 | **多渠道化**：不依赖单一微信生态；同时运营小红书+知乎+即刻+Twitter |
| 支付渠道受阻 | 中 | 高 | 小报童（微信）+ 爱发电（支付宝）双渠道；Stripe备用 |
| 被当作割韭菜 | 中 | 高 | 免费试看版透明展示内容质量；收益测算保守；标注风险 |
| 内容枯竭 | 低 | 中 | 内容生成器可持续产出；4周缓冲库存 |
