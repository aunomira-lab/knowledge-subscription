## 🚀 快速开始

### 每日运营流程

```bash
# 1. 进入项目目录
cd /home/AgentAdmin/.hermes/shared/dev-team/projects/knowledge-subscription

# 2. 更新今日数据 (填写实际数据)
python3 scripts/update_metrics.py \
  --date=$(date +%Y-%m-%d) \
  --visits=50 \
  --signups=10 \
  --paid=1 \
  --revenue=30 \
  --title="今日内容标题"

# 3. 验证数据正确性
python3 scripts/validate_experiment.py

# 4. 查看实验摘要
python3 scripts/update_metrics.py --summary
```

### 关键命令速查

```bash
# 查看最新数据
tail -1 metrics/experiment_tracker.csv

# 统计累计收入
awk -F',' 'NR>1 {sum+=$9} END {print "总收入: ¥"sum}' metrics/experiment_tracker.csv

# 查找收入非零的日期
csvgrep -c daily_revenue -m "[^0]" metrics/experiment_tracker.csv
```

### 停止/加码/转向标准

**🛑 停止条件**:
- 7天0付费用户 → 转向免费模式
- 平均打开率<15% → 重新定位内容
- 获客成本>¥100/人 → 暂停付费推广

**⬆️ 加码条件**:
- 3天内3+付费 → 启动周更
- 平均打开率>50% → 推广预算¥200/周
- 3+用户主动分享 → 上线邀请系统

**🔄 转向条件**:
- 有流量无转化 → 延长试用期
- 有转化无留存 → 调整内容形式
- 平台限流 → 迁移至独立站+邮件
