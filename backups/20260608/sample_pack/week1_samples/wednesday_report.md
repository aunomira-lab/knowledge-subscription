## 🤖 AI优化工作流

### 步骤1: 信息收集
输入原简历 + 目标岗位JD + 个人特点

### 步骤2: AI诊断
提示词模板：
```
你是一位经验丰富的HR专家，请分析以下简历：

【简历内容】
[paste 简历全文]

【目标岗位】
[paste 招聘要求]

请给出：
1. 严重问题3个（会导致筛选失败）
2. 中等问题3个（影响面试机会）
3. 优化建议具体措施
4. 匹配度评分 (0-100)
```

### 步骤3: AI重写
根据诊断结果，分模块让AI重写：
- 个人信息模块
- 工作经历模块（STAR法则）
- 项目经验模块（数据化表达）
- 技能特长模块

### 步骤4: 人工精厠工
- 确保数据真实
- 调整语气流畅度
- 统一格式美观
- 校对无语病

### 步骤5: 格式输出
提供：PDF版 + Word编辑版 + 针对性模板

## 📈 问题类型与解决方案

### 问题1: 内容空泛
**表现**: "负责某某工作"、"协调各部门"
**解决**: 改为 "负责XX项目，实现了XX效果，数据提升XX%"

### 问题2: 结构混乱
**表现**: 时间倒序、经历跳槽、重点不突出
**解决**: 重新排版，按时间倒序，重点经历置顶

### 问题3: 和岗位不匹配
**表现**: 简历很好，但和投递岗位无关
**解决**: 针对性调整，突出相关经验

### 问题4: 过度美化
**表现**: 花哨模板、颜色过多、正文内容不足
**解决**: 简洁专业模板，内容为王
