## 💼 AI账号运营工作流

### 阶段1: 账号起号 (0-1万粉)

**账号定位**
用AI分析平台热点：
```
提示词："分析近30天抖音热点，给出涨粉最快的5个细分领域和内容策略"
```

**内容生产**
- AI生成脚本：根据热点扩写
- AI配音: 魔音工坊/剪映AI配音
- AI字幕: 自动识别并生成字幕
- AI封面: 可灵/Midjourney生成吸睛封面

**发布策略**
- 频率: 每日1-3条
- 时间: 7-9点，12-14点，18-22点
- 标签: AI生成热点标签组合

---

### 阶段2: 流量放大 (1-10万粉)

**数据分析**
用AI分析账号数据：
```
分析近30天视频数据，找出：
1. 播放量最高的视频特点
2. 点赞/评论率最高的类型
3. 转化率最好的内容方式
4. 发布时间与流量关系
```

**内容迭代**
- 复制爆款模板
- 稳定更新频率
- 增加互动设计
